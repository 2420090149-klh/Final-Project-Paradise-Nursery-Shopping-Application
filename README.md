A small React frontend for the "Paradise Nursery" shopping app.

Features
- Landing page with background and Get Started button
- Product listing with at least 6 houseplants in multiple categories
- Shopping cart with add/remove/increase/decrease per-item quantity
- Cart count badge in header updates dynamically
- Cart persisted to `localStorage`

Run (Windows PowerShell):

```powershell
# install deps
npm install
# start dev server
npm run dev
```

Build & preview:

```powershell
npm run build
npm run preview
```

The app is a Vite + React project. Open `http://localhost:5173` after running `npm run dev`.
